# 第２章 名前に情報を詰め込む

**鍵となる考え**<br>
名前に情報を詰め込む。

- 明確な単語を選ぶ
- 汎用的な名前は避ける（あるいは、使う状況を選ぶ）
- 抽象的な名前よりも具体的な名前を使う
- 接尾辞や接頭辞を使って情報を追加する
- 名前の長さを決める
- 名前のフォーマットで情報を伝える
