# 第２章　なぜ関数型プログラミングなのか

### 参照透過性を保証する

次のような場面で参照透過性を喪失しました

- 値の参照透過性を破壊する可変なデータ
- 変数の参照透過性を破壊する代入操作
- 関数の参照透過性を破壊する副作用
